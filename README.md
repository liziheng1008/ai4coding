# 个人网站项目

这是一个使用HTML、CSS和JavaScript创建的响应式个人网站。

## 功能特点

- 响应式设计，适配各种设备
- 平滑滚动导航
- 项目展示区域
- 联系表单
- 社交媒体链接

## 部署到Vercel

### 方法1：通过GitHub部署（推荐）

1. 将代码上传到GitHub仓库
2. 访问 [vercel.com](https://vercel.com)
3. 使用GitHub账号登录
4. 点击"New Project"
5. 导入你的GitHub仓库
6. Vercel会自动部署你的网站
